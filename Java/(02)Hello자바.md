# Hello 자바

### 자바코드가 실행되는 과정은 무엇인가?

- 우선 소스코드는 컴파일을 거쳐서 프로그램코드를 컴퓨터가 이해할 수 있도록 엮어주는 작업을 한다.
- 컴파일을 하면 .java파일이 .class파일을 디스크에 저장한다.
- .class파일은 바이너리 파일로 되어있으며 JVM에서 읽어서 운영체제에서 실행된다./

### main 메소드는 무엇인가?

- 자바 프로그램이 시작될때 가장 먼저 실행되는 메소드이다.

### 메소드의 구성요소는 무엇인가?

- 제어자, 리턴타입, 메소드 이름, 매개변수 목록, 예외목록, 메소드 내용 이다.
